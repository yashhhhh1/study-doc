# Getting Started with Create React App

## Client start using  Follow the 3 Steps and your good to go.  

1: Clone the Project 
   ```bash 
      git clone link
   ``` 

2: simply run the command  
   ```bash 
      npm install
   ``` 
   First to install all the packages
   
3: run the project using 
  ```bash 
      npm run dev
   ``` 
   
   ---
